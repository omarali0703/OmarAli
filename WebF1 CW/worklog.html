Starting a web server with WINSCP

  • Connect to your VM using SSH (Secure Shell – e.g. PuTTY or the Chrome plugin).
  • Select the “me” folder (where you have already uploaded your web page):
    cd me↵
  • Run the Web server
    http-server↵
    Ignore the IP addresses that the web server prints out.
  • Now open a browser and enter the IP address of your machine in the address bar.

    Note that the server prints out a couple of IP addresses that won't work. You need to use the IP address shown on http://port.ac.uk/myvm!

    If you see the page you made, congratulations, you’ve just built and deployed a web site.

    If you don’t see your page, try and work out where and why it’s failing, and fix it!

    You can always ask your tutor for pointers.
